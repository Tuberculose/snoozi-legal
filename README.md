- `https://tuberculose.github.io/snoozi-legal/privacy/`
- `https://tuberculose.github.io/snoozi-legal/delete-account/`

## Activation GitHub Pages gratuite

1. Creer un depot public GitHub nomme `snoozi-legal`.
2. Pousser ce dossier dans le depot.
3. Dans GitHub, ouvrir `Settings > Pages`.
4. Dans `Build and deployment`, choisir `GitHub Actions`.
5. Lancer le workflow `Deploy Legal Pages` ou pousser sur `main`.
